# CSE 102: Computer Programming

This repository contains all the homework assignments completed for the CSE 102 Computer Programming course at Gebze Technical University. The course provides a foundational understanding of programming principles using the C language, covering everything from basic syntax and memory management to complex data structures and algorithms.

## Course Summary

The main learning objectives of this course include understanding algorithms, mastering fundamental programming concepts like control structures and functions, and gaining practical experience with advanced topics such as arrays, pointers, file operations, structs, dynamic memory, and linked lists. Each assignment below targets a specific set of these learning outcomes.
