# CSE241 - Object Oriented Programming

## Course Description

This repository contains the projects and assignments for the CSE241 Object Oriented Programming course. The course covers the fundamental principles of object-oriented programming (OOP) using C++ and provides an introduction to Java. Key topics include classes, objects, inheritance, polymorphism, templates, and the Standard Template Library (STL).

## Course Summary

### Topics Covered

- Introduction to C++ programming, classes, and objects
- Control Structures, Functions, and arrays
- Pointers and Strings
- Classes and Data Abstraction
- Operator Overloading
- Inheritance
- Virtual Functions and Polymorphism
- Templates
- Exception Handling
- Standard Template Library (STL)
- Introduction to Java and comparison with C++

### Grading

- **Midterm Exam:** 30%
- **Final Exam:** 30%
- **Programming Assignments:** 30%
